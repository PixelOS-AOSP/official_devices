# Clean flash:
- Flash boot image (recovery) or OrangeFox recovery
- Reboot to recovery
- Sideload ROM zip
- Format data
- Reboot to system & Enjoy

# Updating to a newer build (dirty flash):
- Update via OTA Updater, or
- Reboot to Recovery
- Flash PixelOS
- Flash Magisk (Optional)
- Reboot to system
