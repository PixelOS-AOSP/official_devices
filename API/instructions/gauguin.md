# Clean flash (Required from Android 13 to Android 14):
- Flash the recovery
- Download the ROM
- Reboot to recovery
- Flash the latest MIUI 14 firmware for your region
- Flash ROM zip
- Format data
- Reboot and voila!

# Updating to a newer build (dirty flash):
- Update via OTA Updater, or
- Flash ROM zip (sideload)
- Reboot and voila!
