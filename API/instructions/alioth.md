# Clean flash:
- Flash boot image (recovery) or OrangeFox recovery
- Reboot to recovery
- Flash MIUI 13.0.3 Global Stable Firmware
- Sideload ROM zip
- Format data
- Reboot and voila!

# Updating to a newer build (dirty flash):
- Update via OTA Updater, or
- Flash ROM zip (sideload)
- Reboot and voila!
