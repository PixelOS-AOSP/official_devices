# Clean flash:
- Download ROM from the link above
- Reboot to recovery
- Flash the latest MIUI 13.0.2.0 firmware
- Flash ROM zip
- Format DATA
- Reboot and voila!

# Updating to a newer build (dirty flash):
- Update via OTA Updater, or
- Flash ROM zip (sideload or USB-OTG)
- Wipe Cache & Dalvik Cache
- Reboot and voila!
