# Clean Flash
- Reboot to recovery.
- Wipe System, Vendor, Dalvik/Cache, Data.
- Flash ROM.zip, Magisk (Optional)
- Reboot

# Dirty Flash

- Place the .zip wherever you like
- Flash ROM.zip and magisk zip (optional) via TWRP recovery
- Reboot and voila!
